Как пользоваться калькулятором?

Калькулятор очень прост в использовании:

• Нажимайте на цифры для ввода чисел
• Используйте кнопки операций (+, -, x, /) для выбора действия
• Нажмите "=" для получения результата
• Кнопка "C" очищает все значения
• Кнопка "+/-" меняет знак числа

Какие математические операции поддерживаются?

Калькулятор поддерживает четыре основные операции:

• Сложение (+)
• Вычитание (-)
• Умножение (x)
• Деление (/)

В будущем планируется добавить проценты и другие функции.
